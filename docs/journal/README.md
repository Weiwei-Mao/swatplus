# Learning Journal

The journal preserves the chronological learning process. Create one entry per focused session under the matching year using `YYYY-MM-DD-short-topic.md`.

Journal entries may include failed breakpoints, incomplete interpretations, and questions, but every statement must be labeled as an observation, hypothesis, or verified result. When a result becomes durable, link to the canonical note or trace where it was promoted.

Use the [journal template](../templates/journal-entry.md).
