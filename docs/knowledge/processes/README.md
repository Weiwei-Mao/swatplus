# Processes

Scientific processes and state transitions belong here: hydrology, soil water, erosion, plant growth, nutrients, carbon, management, pesticides, pathogens, salts, and related calculations.

Create subfolders only after a process has more than a few focused notes. Every equation note must identify units, initialization/reset timing, inputs, outputs, callers, and downstream mass-balance effects.

Do not infer the active process only from a routine name; link the control path or trace that proves it is selected for the scenario.
